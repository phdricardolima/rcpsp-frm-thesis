# Scripts Python

Adicione nesta pasta somente os scripts finais efetivamente utilizados no pós-processamento dos resultados da tese.

Estrutura recomendada:

```text
python/
├── analysis/       Estatísticas confirmatórias, ponderações e auditorias
├── figures/        Figuras do capítulo de resultados e dos apêndices
├── tables/         Tabelas finais exportadas
├── requirements.txt
└── README.md
```

Para cada script, registre:

1. nome e função científica;
2. arquivos de entrada obrigatórios;
3. arquivos de saída produzidos;
4. parâmetros de linha de comando;
5. versão do Python e dependências;
6. relação com a tabela ou figura da tese.

Não mantenha versões intermediárias, cópias numeradas ou scripts não utilizados na versão final da tese.
